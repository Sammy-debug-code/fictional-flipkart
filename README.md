# BrightMart Retail Analytics – Power BI Project

## Project Overview

This repository contains a **Power BI analytics project built using the BrightMart retail dataset**. The goal of this project is to analyze retail sales performance and extract meaningful business insights using **data modeling, DAX calculations, and interactive dashboards**.

The project explores key retail metrics such as **revenue, gross profit, marketing ROI, and inventory performance** while identifying sales trends across different product categories and time periods.

This project is part of my **data analytics learning journey**, where I am building hands-on projects to strengthen my skills in **Power BI, data analysis, and business intelligence.**

---

## Key Business Questions

The analysis aims to answer questions such as:

* Which **product categories generate the highest revenue and profit?**
* Are there **seasonal patterns in sales performance?**
* How effective are **marketing campaigns in generating revenue?**
* Which categories contribute the most to **overall profitability?**

### DAX Calculations

Examples of measures used in the project:

* Revenue
* Gross Profit
* Marketing ROI
* Inventory Metrics (Average Inventory)

Example:

```DAX
ROI = DIVIDE(([Revenue] - [Total Marketing Spend]), [Total Marketing Spend], 0)
```
